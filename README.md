# ⏰ My-Alarm

A powerful Android alarm clock app with smart wake-up challenges, sleep tracking, and customizable alarms.

## 📱 Features

### Core Alarm Features
- ✅ Set multiple alarms with custom times
- ✅ Repeat alarms on specific days
- ✅ AM/PM time selection

### Smart Wake-Up Challenges
- 🧮 **Math Problems** - Solve equations to turn off alarm

### Customizable Sound
- 🎵 **Select Local Music** - Choose any song from your device
- 🔔 **Default Alarm Sounds** - Pleasant built-in sounds
- 🔊 **Gradual Volume Increase** - Wakes you gently

### Guardian Angel (Fail-Safe)
- 👼 **Automatic Activation** - Kicks in if you don't wake up
- 📈 **Progressive Volume** - Gets louder over time
- 🚨 **Emergency Sound** - Final wake-up attempt

### Sleep Tracking
- 📊 **Sleep Quality Analytics** - Track your sleep patterns
- 📈 **Wake-up Success Rate** - Monitor your progress

### Timer
- ⏱️ **Countdown Timer** - Set timers for cooking, studying, etc.
- 🎯 **Timer History** - Track your timer usage

### Additional Features
- 🌤️ **Weather Integration** - See current weather when alarm rings
- 📅 **Calendar Sync** - Auto-create alarms from calendar events
- 👤 **Profile System** - Save different alarm profiles (Work, Weekend, Travel)
- 📊 **Detailed Analytics** - Track your wake-up patterns
